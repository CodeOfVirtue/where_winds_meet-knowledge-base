# where_winds_meet-knowledge-base
Archive of community-derived WWM resources (damage formula sheets, penetration notes, CN sources, maps) with changelog and attribution.

# Disclaimer

This repository is a community-driven research and reference project.

- Damage formulas, spreadsheets, and interpretations are derived from publicly available sources and community testing.
- Some mechanics (e.g. "external defense") are **inferred** and not exposed directly in-game.
- While the information here is believed to be accurate based on current understanding, it is **not guaranteed to be complete or final**.
- Game updates, balance changes, or newly discovered mechanics may invalidate parts of this material.

Use this information as a reference, not as official documentation.
